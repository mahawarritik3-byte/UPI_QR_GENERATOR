💳 UPI QR Code Generator

A simple Python project to generate a universal UPI QR code that works with apps like:

Google Pay
PhonePe
Paytm


Users can scan the generated QR code and make payments directly through any UPI-supported application. 🚀

✨ Features

✅ Generate universal UPI QR code

✅ Dynamic UPI ID input

✅ Save QR code as PNG image

✅ Automatically open QR code

✅ Beginner-friendly Python project

🛠️ Tech Stack

🐍 Python

📦 qrcode Library

🖼️ Pillow (PIL)

📁 Project Structure

project/

├── main.py

├── upi_qr.png

└── README.md
